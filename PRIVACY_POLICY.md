Privacy Policy

Where is the Question? does not collect, transmit, or share any personal or sensitive user data.

All data used by the extension is stored locally on the user’s device using Chrome’s storage API and is never transmitted to external servers. This data is limited to question text and positional information required to provide the extension’s core functionality.

The extension only runs on specified ChatGPT domains and does not access data from other websites.

No third-party services, analytics tools, or remote code are used.

If you have any questions, please contact the developer.
